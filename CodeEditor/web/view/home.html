Code Editor

CodeEditor is a web application that helps programmer to create and see the result of the executed program by terminal, real-time collaboration with other programmers by chat or invite to join the same file and manage the code such as import, export, shared files. Currently CodeEditor only supports C, C ++, and Java programming languages. CodeEditor has main features: provide workspace to make, execute and build the source code, real-time collaboration, chat, and build the terminal.

What is CodeEditor?

Web application that provides workspace to create the program code, perform, display the results of the code through the terminal, and collaborate with other users in real-time.

Why Use It?

Real-time code editor, chat with other programmers, see your program results in terminal, download and upload your files. All integrated in this application.
